Lưới bootstrap
        Lưới bootstrap gồm tối đa là 12 cột

        Tạo ra các giao diện có số cột mà độ rộng bằng nhau
        ví dụ
        1 cột => col-md-12
        2 cột => col-md-6
        3 cột => col-md-4
        4 cột => col-md-3
        6 cột => col-md-2
        12 cột => col-md-1

        công thức : suffix trong col-md-?
        12/số cột = suffix = ?
